# Link Strategy

## Link Types (Choose One Per Post)

### Option A: Direct Affiliate Link
Use when promoting a specific product directly:
```
Amazon affiliate link format: https://www.amazon.com/dp/ASIN?tag=syndicate-20

Example: https://www.amazon.com/dp/B07XYZ1234?tag=syndicate-20
```

### Option B: Site Link (Preferred for SEO + Multiple Affiliate Opportunities)
Use when driving traffic to guides with multiple product recommendations:
```
Guide link: https://vestaveranda.com/guides/[guide-slug]

Examples:
- https://vestaveranda.com/guides/fire-pit-composite-deck
- https://vestaveranda.com/guides/robot-mower-hills
- https://vestaveranda.com/guides/backyard-lighting-no-electricity
```

## When to Use Each

| Scenario | Link Type | Why |
|----------|-----------|-----|
| Single product highlight | Direct Affiliate | Higher conversion |
| How-to/guide content | Site Link | Multiple products, SEO boost |
| Comparison posts | Site Link | Visitors can choose |
| Quick tips | Site Link | Builds site traffic |
| Product roundups | Site Link | Multiple affiliate opportunities |
